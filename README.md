# Portfolio: Alexandre de Paula

Bem-vindo ao repositório do meu portfólio profissional, reestruturado como uma SPA (Single Page Application) focada em conversão, apresentação de cases e ofertas de soluções em Engenharia de Software.

## 🎯 Objetivo
Desfocar do formato tradicional de "currículo web" para uma estrutura moderna B2B. O sistema visa atrair potenciais clientes mostrando resultados práticos de automações, SaaS e e-commerces que geraram lucro e performance.

## 🛠️ Stack Tecnológica
*   **Motor:** Vite + React (TypeScript) para inicialização instantânea.
*   **Estilização:** Tailwind CSS v4, adotando a nova API `@theme` e simplificando o empacotamento.
*   **Micro-Animações:** Framer Motion, gerando fluidez na UX ao interagir com seções e rolagem da página.
*   **Boas Práticas:** Os componentes não ultrapassam 200 linhas. O modelo é orientado a separação de Interface (UI) e Lógica Custom (Hooks/Dados).

## 🎨 Visual & UI/UX
O design apresenta tipografia legível combinada com elementos flutuantes (Glassmorphism). 
- Há um suporte ao **Modo Claro** e **Escuro**, acompanhado de uma lógica algorítmica de horas. Quando acessado após as 18h corporativamente, o site adota a experiência escura para conforto visual.
- A paleta de cores se expande a um fundo flexível (`#EDEDED` vs `#1A1A1A`) e detalhes contrastantes baseados em Azul Turquesa/Ciano (`#28D2F8`) e Roxo (`#BC32D4`).

## 🚀 Como Executar

A aplicação está configurada com suporte retrocompatível e atual build pipeline:

```bash
# Inicializar Dependências
npm install

# Rodar Ambiente de Desenvolvimento Local
npm run dev

# Fazer Build Estático 
npm run build
```
